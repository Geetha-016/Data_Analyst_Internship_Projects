# Crowdfunding Campaign Analysis (Kickstarter Dataset)

## Project Overview
Kickstarter is a popular crowdfunding platform for creative projects. This project analyzes Kickstarter campaigns across multiple years to identify patterns in project success, funding behavior and category performance.

The analysis was carried out using Excel, Tableau, Power BI and SQL, with a focus on building interactive dashboards and extracting actionable business insights.

---

## Objectives
- Understand factors contributing to successful crowdfunding campaigns  
- Compare project outcomes across categories, locations and funding goals  
- Analyze funding trends and backer behavior  
- Visualize insights using multiple BI tools  
- Present findings in an interactive, business-friendly format  

---

## Dataset
The dataset contains information about crowdfunding campaigns, including funding amount, backers, category, location, goals, outcomes and timelines.

Due to GitHub file size limitations, the dataset is hosted externally.

## Tools & Technologies
- Excel (Data Cleaning & Dashboarding)  
- Tableau (Data Visualization)  
- Power BI (Interactive Dashboard & Data Modeling)  
- SQL (Data Querying & Data Extraction)  

---

## Key Insights & Findings
- Over 51% of projects failed  
- Product Design was the top-performing category  
- Projects with goals under $10K were ~3x more likely to succeed  
- New York, Los Angeles and London had the highest project counts  
- Peak activity observed between 2014–2016  

---

## Challenges Faced
- Data cleaning due to inconsistent category and location values  
- Handling date fields and creating hierarchies across tools  
- Performance optimization in Power BI using slicers  
- Managing differences across Excel, Tableau and Power BI  

---

## Tool Comparison
- Excel: Simple but mostly static  
- Tableau: Strong visuals but learning curve in filters  
- Power BI: Best interactivity for business use  

---

## Interpretation
Crowdfunding success is influenced by funding goals, category type and location. Lower funding goals and strong community engagement increase the chances of success.

---

## Dashboard Features
- Key metrics: Amount Raised, Backers, Average Duration  
- Outcome distribution (Successful, Failed, etc.)  
- Category and location analysis  
- Time trends (Year & Month)  
- Top projects by funding and backers  
- Success rate by goal range
